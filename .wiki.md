# Project Summary
Huginn is a zero-knowledge ephemeral group chat application designed for secure, encrypted communications. It leverages WebRTC technology to facilitate peer-to-peer messaging, ensuring that messages are encrypted in the user's browser and automatically deleted after the session ends. This serverless architecture caters to privacy-conscious users, enabling effortless room creation, unique URL sharing, and a seamless messaging experience. Recent updates have enhanced the footer's background color for better visibility.

# Project Module Description
Huginn consists of the following functional modules:
- **Chat Interface**: Facilitates real-time message exchange.
- **Participant Management**: Displays participants and their connection statuses.
- **Security Status**: Indicates the encryption status of the chat.
- **Room Management**: Supports room creation with options for custom names and passwords, now featuring an improved UI.

# File Description Inventory
- **App.tsx**: Main application component that sets up routing and state management.
- **Room.tsx**: Interface for chat rooms, handling messages and participant interactions.
- **Index.tsx**: Updated landing page for creating or joining chat rooms with improved UI for room options and branding.
- **SecurityStatus.tsx**: Displays the current security status of the chat.
- **ChatMessage.tsx**: Renders individual chat messages.
- **ParticipantList.tsx**: Shows a list of participants in the chat room.
- **crypto.ts**: Contains cryptographic utilities for message encryption and decryption.
- **webrtc.ts**: Manages WebRTC connections for peer-to-peer messaging.
- **room.ts**: Handles room creation and management functionalities, including new room options.

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the application:
   ```
   pnpm run start
   ```
