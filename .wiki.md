# Project Summary
Huginn is a zero-knowledge ephemeral group chat application that prioritizes secure, encrypted communications. Utilizing WebRTC technology for peer-to-peer messaging, it ensures that messages are encrypted in the user's browser and automatically deleted after the session ends. This serverless architecture is designed for privacy-conscious users, enabling easy room creation, unique URL sharing, and a streamlined messaging experience. The recent updates include a rebranding effort that features a new logo and enhanced user interface design.

# Project Module Description
Huginn comprises the following functional modules:
- **Chat Interface**: Facilitates real-time message exchange.
- **Participant Management**: Displays participants and their connection statuses.
- **Security Status**: Indicates the encryption status of the chat, now featuring the new logo.
- **Room Management**: Supports room creation with options for custom names and passwords.
- **Cookie Consent**: Displays a popup for cookie consent, informing users that their information is not shared with third parties.

# File Description Inventory
- **App.tsx**: Main application component that sets up routing and state management, including the Cookie Consent component.
- **CookieConsent.tsx**: Displays a cookie consent popup, allowing users to accept or decline cookies.
- **Room.tsx**: Interface for chat rooms, handling messages and participant interactions.
- **Index.tsx**: Updated landing page for creating or joining chat rooms, featuring a centered logo and improved UI for room options and branding.
- **SecurityStatus.tsx**: Displays the current security status of the chat, now using the new logo.
- **ChatMessage.tsx**: Renders individual chat messages.
- **ParticipantList.tsx**: Shows a list of participants in the chat room.
- **crypto.ts**: Contains cryptographic utilities for message encryption and decryption.
- **webrtc.ts**: Manages WebRTC connections for peer-to-peer messaging.
- **room.ts**: Handles room creation and management functionalities.

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the application:
   ```
   pnpm run start
   ```
