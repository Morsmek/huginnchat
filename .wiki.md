# Project Summary
Huginn is a zero-knowledge ephemeral group chat application designed for secure, encrypted communication. Utilizing WebRTC for peer-to-peer messaging, it ensures that all messages are encrypted in the user's browser and destroyed after the session ends, providing complete privacy without the need for server-side storage.

# Project Module Description
Huginn consists of several functional modules:
- **Chat Interface**: Allows users to send and receive messages in real-time.
- **Participant Management**: Displays the list of participants and their connection status.
- **Security Status**: Indicates the encryption status of the chat.
- **Room Management**: Handles room creation and URL parsing for joining existing chats.

# File Description Inventory
- **App.tsx**: Main application component that sets up routing and state management.
- **Room.tsx**: Chat room interface where users can send and receive messages.
- **Index.tsx**: Landing page for creating or joining chat rooms.
- **SecurityStatus.tsx**: Displays the current security status of the chat.
- **ChatMessage.tsx**: Renders individual chat messages.
- **ParticipantList.tsx**: Displays a list of participants in the chat room.
- **crypto.ts**: Contains cryptographic utilities for message encryption and decryption.
- **webrtc.ts**: Manages WebRTC connections for peer-to-peer messaging.
- **room.ts**: Handles room creation and management functionalities.

# Usage
1. Install dependencies:
   ```
   pnpm install
   ```
2. Build the project:
   ```
   pnpm run build
   ```
3. Run the application:
   ```
   pnpm run start
   ```
